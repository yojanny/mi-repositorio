Elementos de bloque vs elementos de linea

elementos de bloque

Los elementos de bloque generan saltos de linea.no tienen a los lado,el resto del contenido se colocan encima o abajo segun el orden del html

Dentro de un bloque podemos meter practicamente cualquier otra etiqueta de bloque o de linea

Ejemplos: header, main, footer, setion, article, aside, div, p, h1,h2,h3,h4,h5,h6,

elementos de linea

los elementos ocupan lo minimo
